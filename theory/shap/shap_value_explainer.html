Baseline — the average model prediction across all data
Baseline = 38%
If you know nothing about a passenger, the model guesses 38% survival chance (the dataset average). SHAP values measure how each feature moves this number.
The key formula
prediction = baseline + Σ SHAPi
Every prediction is just the baseline shifted by the sum of all SHAP values. They must add up exactly — no rounding, no approximation.

SHAP values have 3 mathematical guarantees:
1. Local accuracy
SHAP values always sum exactly to (prediction − baseline). No leftover, no approximation.
87% − 38% = 0.28 + 0.14 + 0.05 + 0.04 = 0.49 ✓
2. Consistency
If a feature contributes more in model B than model A (for every possible coalition), it always gets a higher SHAP value in B. The ranking is fair.
3. Missingness
A feature that has zero effect on any prediction always gets a SHAP value of exactly 0.
useless_feature SHAP = 0.00
